Building Overview

The X-Bow is an extremely powerful single target defensive building. The tower can be toggled between attacking ground troops or ground and air troops. If it’s set to attack ground troops it will get a boost in the range. If it’s set to attack ground and air units it will have a medium range. The tower uses elixir as its ammo so it will need to be reloaded with more elixir when it runs out.

Building Information

Damage type: Single target

or

Range: 11 tiles

Building Models

The graphic models for each level of the X-Bow are shown below. The base of the tower is what changes at every level and the bow part of the tower will remain the same. Level one looks like it’s using stone and a dark steel to create the base. Level two uses dark steel and gold for the bottom of the tower. Level three has a very dark base and consists of black crystals and a very dark steel. The fourth level of the tower is similar to the third level. The fourth uses red crystals and dark steel for the base.
